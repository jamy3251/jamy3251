## About Me

- 정보통신공학을 기반으로 **AI, 백엔드, 클라우드, 보안, 시각화 인터페이스**를 함께 확장해가고 있는 개발자입니다.
- 단순히 기능을 구현하는 개발자가 아니라, **사람이 실제로 쓰고 도움을 받을 수 있는 제품**을 설계하고 만드는 개발자가 되고자 합니다.
- 공모전, 팀 프로젝트, 해커톤, 부트캠프 경험을 통해 아이디어를 빠르게 구조화하고, 협업 속에서 **기획부터 구현까지 연결하는 방식**을 배워왔습니다.
- 특히 “무엇을 어떻게 배워야 할지 막막한 문제”에 관심이 많아, **개인 맞춤형 학습 로드맵, 과제 생성, 진척 관리** 같은 AI 기반 학습 시스템을 직접 만들고 있습니다.
- 저는 화려한 결과보다 먼저 **좋은 정보 구조, 명확한 사용자 흐름, 유지보수 가능한 설계**를 중요하게 생각합니다.
- 앞으로는 AI, 서버, 데이터, 제품 설계를 더 깊게 결합해, **교육·생산성·실생활 문제를 해결하는 서비스**를 만드는 것이 목표입니다.
